# Network Tools

This is a container image that contains a set of network tools that can be used for troubleshooting and debugging network issues.

## Tools included

- `ping`
- `traceroute`
- `nslookup`
- `dig`
- `netstat`
- `tcpdump`
- `curl`
- `wget`
- `iperf3`
- `nmap`
- `mtr`
- `telnet`
- `ssh`
- `sftp`
- `ftp`
- `nc`
